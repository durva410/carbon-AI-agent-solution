# Data Collection Layer

## Purpose
To measure real-time energy and CO₂ emissions generated by AI/ML workloads.

## Tools Used
- CodeCarbon
- psutil
- nvidia-smi (GPU stats)
- Prometheus exporters
- Docker containers

## Metrics Tracked
- Runtime duration
- GPU/CPU utilization
- Power consumption (W)
- Energy (kWh)
- CO₂ emissions (kg CO₂e)

## Output
Stores collected metrics in SQLite/PostgreSQL for agent analysis.
